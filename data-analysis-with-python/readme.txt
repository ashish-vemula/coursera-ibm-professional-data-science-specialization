Hand-On exercises for "Data Analysis with Python" course.

***reveiw-introduction - Basic insights of data sets

***data-wrangling - Handling missing data, Data formating, Data normalization (simple-feature scaling, Min-Max, Z-Score), Binning,                             Converting categorical variables to numerical values (one-hot encoding)

***exploratory-data-analysis - Analyzing individual feature patterns using visualization (regplot, boxplot), Descriptive statistical                                      analysis, Basics of grouping (pandas pivot, heatmap visualization), Correlation (Pearson correlation),                                    ANOVA (F-test_score, p-value)

***model-development - Model Development (Linear Regression, Multiple Linear Regression), Model Evaluation Using Visualization (Regression                        Plot, Residual Plot, Distribution Plot), Polynomial Regression and Pipelines, Measures for In-Sample                                      Evaluation(R^2/R-Squared, MSE)
